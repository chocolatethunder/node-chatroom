Node.js client built using:
- Node
- Express
- Jade
- Socket.io

This code was built with the help of this
tutorial https://socket.io/get-started/chat/
Some parts of the code may match.

## How to Run ##

#### Clone the repository 
```
npm install
```
```
npm run start
```

#### If you want to run nodemon
```
npm run devstart
```
